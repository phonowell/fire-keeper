## Project
- Name: fire-keeper

## Summary
- EN: Pure ESM Node.js/TypeScript filesystem automation toolkit for backup, copy, remove, glob, watch, and concurrent workflows.
- ZH: 纯 ESM 的 Node.js/TypeScript 文件系统自动化工具库，支持备份、复制、删除、匹配、监听和并发执行。

## What it does / 主要用途
- EN: Build/deploy file workflows, batch file operations
- ZH: 构建部署文件流转，批量文件处理

## Keywords / 关键词
- EN: nodejs, typescript, filesystem, file-operations, automation, esm, cli, glob, file-watcher, concurrency
- ZH: Node.js, TypeScript, 文件系统, 文件操作, 自动化, ESM, 命令行, glob, 文件监听, 并发

## Install / 安装
- EN: pnpm i fire-keeper
- ZH: pnpm i fire-keeper

## Quickstart / 快速开始
```bash
pnpm i fire-keeper
node --input-type=module -e "import backup from 'fire-keeper/backup.js'; await backup('./config.json')"
```
