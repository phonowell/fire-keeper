# Fire Keeper

一个简单的项目构建工具。

## 安装

```shell
pnpm i fire-keeper
```

## 使用

首先，在代码中引入模块。

```typescript
import $ from 'fire-keeper'
```

之后就可以愉快地玩耍啦。

```typescript
$.log('hello world')
```

## 构建

```shell
npm run build
```

## 测试

```shell
npm run test
```
